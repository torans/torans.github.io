【Golang】Mac下编译可以在linux下运行的程序

在mac下直接使用go build编译出来的文件在linux下无法运行，因此需要使用其他方法编译，执行下面的命令即可：

 CGO_ENABLED=0 GOOS=linux GOARCH=amd64 go build main.go

该命令会下载相应的包,然后可以看到编译成功。
