NVIDIA NVML Driver/library version mimatch 解决办法

June 24, 2022

###问题描述：执行nvidia-smi命令的时候出现一下错误：

root@localhost:/mnt/md0/lotusWorker# nvidia-smi
Failed to initialize NVML: Driver/library version mismatch

解决办法：卸载原先加载的驱动 

sudo rmmod nvidia_drm
sudo rmmod nvidia_uvm
sudo rmmod nvidia_modeset
sudo rmmod nvidia
# 再次执行nvidia_smi
